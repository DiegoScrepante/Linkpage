🔗 Linkpage

Uma página de links personalizada inspirada no conceito Link-in-bio, que reúne redes sociais, portfólio e outros links importantes em um único lugar.

O projeto foi desenvolvido com foco em prática de desenvolvimento front-end e criação de páginas simples, responsivas e fáceis de compartilhar.

🚀 Funcionalidades

📎 Centralização de links importantes

👤 Exibição de perfil e redes sociais

📱 Layout responsivo para mobile e desktop

🎨 Interface simples e personalizável

⚡ Carregamento rápido por ser uma página estática

🛠️ Tecnologias Utilizadas

HTML

CSS

JavaScript
